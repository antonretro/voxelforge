
# VoxelForge 🛠️

VoxelForge is a high-performance, immersive 3D voxel survival engine built from the ground up for the web. Inspired by the tactical feel of **ArloCraft** and the high-definition aesthetics of the **Igneous** texture pack, it features a native 3D viewmodel system, procedural world generation, and advanced survival mechanics.

## ✨ Key Features

- **3D Native Viewmodel**: A fully animated, 3D blocky arm and held-item system that reacts to movement, mining, and interaction.
- **Igneous Texture Integration**: Optimized for the Igneous pack, featuring HD block textures and destruction overlays.
- **Advanced Physics & Particles**: Real-time 3D particle shards that spawn when mining, with satisfying tactile feedback.
- **Procedural Biomes**: Lush jungles, gnarled swamps, and vast plains generated with multi-layered noise.
- **Survival Systems**: Comprehensive health, hunger, and damage mechanics with a Font Awesome-powered pixelated HUD.
- **Chest & Loot System**: Discoverable treasure chests with randomized loot from a massive 500+ item library.

## 🚀 Getting Started

### Prerequisites
- Node.js
- A modern web browser with WebGL2 support

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/USER_NAME/voxelforge.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Launch the engine:
   ```bash
   npm run dev
   ```

## 🛠️ Built With
- **Engine**: Custom WebGL2 Voxel Engine
- **Logic**: Vanilla JavaScript (ES6+)
- **Styling**: Tailwind CSS & Vanilla CSS
- **Assets**: Igneous Texture Pack
